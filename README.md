# 药物推荐计算器

## 功能介绍
- 基于CatBoost模型的药物推荐系统
- 网页界面，支持输入症状特征并获得药物推荐
- 药物推荐结果按概率从高到低排序
- 支持批量预测和单样本预测

## 安装步骤

### 1. 安装Python
确保您的电脑已安装Python 3.10或更高版本。

### 2. 安装依赖
打开命令提示符，进入本文件夹，执行以下命令：

```bash
pip install -r requirements.txt --user
```

### 3. 运行计算器
执行以下命令启动网页计算器：

```bash
python app.py
```

### 4. 访问计算器
打开浏览器，访问以下地址：
```
http://127.0.0.1:8080
```

## 使用方法
1. 在网页界面上选择或输入症状特征
2. 点击"预测"按钮
3. 查看按概率从高到低排序的药物推荐结果

## 文件说明
- `app.py` - 主应用程序
- `templates/index.html` - 网页界面
- `models/` - 药物预测模型
- `final_input_features.txt` - 输入特征列表
- `output_features.txt` - 输出特征列表
- `requirements.txt` - 依赖包列表
